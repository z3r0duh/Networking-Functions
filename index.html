Networking Concepts Quiz

1. What does a Content Delivery Network (CDN) do?

Caches and delivers data efficiently to users
Encrypts network traffic for secure communication
Provides real-time video streaming to users

2. What is the main purpose of a VPN concentrator?

Prioritizes applications on the network
Handles secure VPN connections and encrypts/decrypts data
Redirects traffic to the fastest network path

3. How does a Time to Live (TTL) field in an IP packet help prevent routing loops?

TTL limits the number of hops a packet can take before being discarded
TTL increases the number of hops for faster delivery
TTL prevents packets from being cached on CDN servers

4. Which protocol configuration helps prioritize real-time traffic like video over file transfers?

VPN
QoS (Quality of Service)
TTL
